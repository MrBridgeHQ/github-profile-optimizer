# Profile README Template

> Placeholder convention: replace every `[placeholder]` with real content. Where a number belongs, use `[metric: provided by user or 'not available']`. Never invent stars, followers, downloads, or dates. Any compatibility or registry claim is marked `verified` or `to verify`, never assumed.

This file gives two ready-to-copy variants of a GitHub profile README (the special `username/username` repository). Copy the body from inside the fenced code block. A metrics block should appear only if the user provides real figures; otherwise leave it out.

Note: an EN/FR bilingual variant can be produced on request. Ask for it explicitly and the two language blocks will be delivered together.
