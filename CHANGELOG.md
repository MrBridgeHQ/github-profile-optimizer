## [0.1.0] - 2026-07-02

### Added

- Initial release of the github-profile-optimizer skill.
- SKILL.md with five modes (Profile Audit, Profile README Rewrite, Repository SEO Audit, Agent Skill SEO & Registry Submission, Skill Positioning).
- References: profile optimization playbook, repository SEO playbook, agent skills registry playbook, registry targets.
- Templates for each mode, three worked examples, an optional public-context collector script, and evals.
